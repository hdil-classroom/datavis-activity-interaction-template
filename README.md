# Data Visualization In-Class Activity

This repository is designed for an in-class activity in the Introduction to
Data Visualization course at Yonsei University.

The dataset is from [https://www.kaggle.com/datasets/kangminjung0405/korean-box-office-20102025-04](https://www.kaggle.com/datasets/kangminjung0405/korean-box-office-20102025-04).
